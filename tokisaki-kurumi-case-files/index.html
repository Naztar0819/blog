Case File Ⅰ 狂三偵探

　　「──咿呀──────────！？」
　　「槍，槍擊！？到底是從哪裡！？」

　　「我的領域……？」
　　狂三微微皺了皺眉。雖然要應付茉莉花很麻煩，但狂三不認為她會毫無意義地說出這番話。
　　「能給我看一下這封威脅信嗎？」
　　「當然！」

　　狂三仔細端詳了一下信封，隨後取出裝在裡面的信，看向了信紙。

　　「五月十八日。
　　栖空辺宅将發生惨劇。
　　魔彈射手」

　　上面用毫無意義的印刷字體寫著這樣的話。
　　這段文字並沒有什麼明顯的內容。無法確定是否是威脅，只是看起來意味深長的一封信。
　　但是，送出這封信的人的名字，讓狂三微妙的在意。

　　「誒？」
　　「我說我接受了。請說一下詳情。」
　　「──！真的嗎！？」

　　狂三半睜著眼說道，隨後茉莉花一副不能接受的模樣大喊起來。

　　◇

　　「…………」
　　偵探狙擊事件之後不久。

　　嘛，話雖如此──
　　「啊哇哇哇哇……！如，如何，如何呢？本小姐的家裡發生大事了！」

　　「沒事。」
